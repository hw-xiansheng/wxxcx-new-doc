      # 二、渲染数据页面

      需求： 页面打开的时候获取购物车数据进行界面渲染，渲染空状态、列表状态、未登录状态

      思路：

      • 通过辅助函数将cart模块里面的相关信息导入进行调用和渲染
      • 在页面中根据需求进行渲染

      实现：

      • 书写辅助函数

        import { mapActions,mapGetters,mapState } from 'vuex'

      methods:{
    // 修改数量
    async changeNum(type, row) {
        let num = row.number
        if (type === 'add') {
            num += 1
        }
        if (type === 'reduce') {
            if (num === 1) return uni.showToast({

            goodsId: row.goods_id, // 商品ID
            id: row.id, // 购物车记录id,
            productId: row.product_id, // 产品ID
            number: num // 数量
        })
        // 重新更新
        this.$store.commit('cart/setCartData', res)
    },
}
